# Wingspan

A small game I work on with [@alexoid](http://twitter.com/alexoid).

* Author: Vaclav Vancura ([@vancura](http://twitter.com/vancura))
* Graphics and music: Alex Schleifer ([@alexoid](http://twitter.com/alexoid))
* Copyright Creeptown Games


## Usage

Use `A-S-D-W` to control left plane, `F` to fire.
Use cursor keys to control right plane, `ALT` to fire.


## Installation

```
npm install
bower install
gulp tsd
gulp
```
